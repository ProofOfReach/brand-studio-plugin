# Typography Principles Reference

## Type Classification (Lupton)

### Serif Families

| Category | Characteristics | Mood | Examples |
|----------|----------------|------|----------|
| **Humanist** | Calligraphic, low contrast, angled stress | Warm, classical, readable | Garamond, Jenson, Palatino |
| **Transitional** | More contrast, vertical stress | Refined, professional | Baskerville, Times, Georgia |
| **Modern/Didone** | High contrast, hairline serifs, vertical stress | Elegant, fashion, editorial | Bodoni, Didot |
| **Slab Serif** | Heavy, block-like serifs | Industrial, bold, sturdy | Rockwell, Clarendon, Courier |

### Sans-Serif Families

| Category | Characteristics | Mood | Examples |
|----------|----------------|------|----------|
| **Grotesque** | Some stroke variation, slightly quirky | Industrial, no-nonsense | Akzidenz Grotesk, Franklin Gothic |
| **Neo-Grotesque** | Uniform stroke width, neutral | Clean, modern, corporate | Helvetica, Arial, Univers |
| **Geometric** | Based on circles/squares | Modern, minimal, tech | Futura, Avant Garde, Century Gothic |
| **Humanist Sans** | Calligraphic influence, varied strokes | Friendly, readable, warm | Gill Sans, Frutiger, Verdana |

### Other Categories

| Category | Characteristics | Best For |
|----------|----------------|----------|
| **Script** | Connected, flowing letters | Invitations, luxury, feminine |
| **Display** | Decorative, attention-grabbing | Headlines, logos, posters |
| **Monospace** | Equal-width characters | Code, typewriter aesthetic |
| **Blackletter** | Gothic, medieval | Traditional, craft, beer |

## Anatomy of Letterforms

Key terms for describing type:

| Term | Description |
|------|-------------|
| **X-height** | Height of lowercase letters (affects readability) |
| **Cap height** | Height of capital letters |
| **Ascender** | Parts rising above x-height (b, d, h, k, l) |
| **Descender** | Parts dropping below baseline (g, j, p, q, y) |
| **Baseline** | Invisible line letters sit on |
| **Serif** | Small strokes at letter terminals |
| **Counter** | Enclosed space within letters (o, e, a) |
| **Bowl** | Curved stroke enclosing counter (b, d, p) |
| **Stem** | Main vertical stroke |
| **Shoulder** | Curved stroke (n, m, h) |
| **Terminal** | End of a stroke without a serif |
| **Stroke contrast** | Difference between thick and thin strokes |

## Type Pairing Principles

### Classic Pairings
- **Serif headline + Sans body**: Editorial, traditional
- **Sans headline + Serif body**: Modern but readable
- **Two weights of same family**: Safe, harmonious
- **Contrast in structure, similar in mood**: Professional

### Pairing Guidelines
1. **Contrast, don't conflict** - Pair different but complementary styles
2. **Limit to 2 families** - More creates visual chaos
3. **Match x-heights** - Ensures visual harmony when mixed
4. **Consider era** - Fonts from similar periods often work together
5. **Weight contrast** - Pair bold with light, not medium with medium

### Avoid
- Two fonts that are too similar (Helvetica + Arial)
- Mixing two decorative/display fonts
- Combining conflicting moods (playful + serious)
- Fonts with incompatible x-heights

## Type Hierarchy

Visual hierarchy through typography:

| Level | Typical Treatment |
|-------|-------------------|
| **Primary (H1)** | Largest, boldest, most distinctive |
| **Secondary (H2)** | Smaller, possibly different weight |
| **Tertiary (H3)** | Subtle differentiation |
| **Body** | Most readable size and weight |
| **Caption** | Smallest, often lighter weight |

**Methods to create hierarchy:**
- Size (most obvious)
- Weight (bold, regular, light)
- Color/contrast
- Spacing (letter-spacing, line-height)
- Case (uppercase, small caps)
- Style (italic, underline)

## Readability Principles

### Body Text
- Sans-serifs work well for digital
- Serifs traditional for print (though debated)
- Optimal line length: 45-75 characters
- Line height: 120-145% of font size
- Adequate contrast with background

### Headlines
- Can sacrifice some readability for impact
- Tighter letter-spacing often works
- Display faces acceptable
- All-caps for short headlines only

### Worst Practices
- ALL CAPS for body text (hardest to read)
- Justified text without hyphenation
- Too little line height
- Too long line length
- Poor contrast

## Typography for Logos

### Wordmark Considerations
- **Custom letterforms** create uniqueness
- **Modified existing fonts** balance uniqueness with familiarity
- **Kerning adjustments** essential for professional look
- **Weight** should match brand personality

### Logo Type Pairings
- Logo type + tagline type should contrast but harmonize
- Primary brand font often differs from logo typeface
- Secondary fonts for body copy can be more neutral

## Variable Fonts (Modern Consideration)

Introduced 2016, allow:
- Single file with multiple weights/widths
- Smooth transitions between styles
- Custom "in-between" weights
- More control with smaller file sizes

## Non-Latin Scripts (Lupton)

When working globally, consider:
- **Arabic**: Right-to-left, connected, cursive
- **Chinese**: Logographic, square proportions
- **Japanese**: Three scripts (Kanji, Hiragana, Katakana)
- **Korean (Hangul)**: Syllabic blocks
- **Indic scripts**: Horizontal headline connector
